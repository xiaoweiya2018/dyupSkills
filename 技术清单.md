# 🧩 二、各模块详细选型

---

## 1️⃣ 抖音视频抓取

### 推荐方案（优先级从高到低）

#### ✅ 方案A：yt-dlp（强烈推荐）

* 支持抖音
* 稳定性高
* 自动解析视频链接

```bash
yt-dlp "抖音视频URL"
```

👉 优点：

* 不需要自己写爬虫
* 社区维护活跃

---

#### ✅ 方案B：Github开源抖音爬虫（备选）

关键词：

* douyin spider
* tiktok crawler

👉 建议选：

* 有cookie支持
* 最近有更新的项目

---

## 2️⃣ 视频 → 音频处理

### 推荐：ffmpeg（必选）

```bash
ffmpeg -i input.mp4 -vn -acodec mp3 output.mp3
```

👉 Python调用：

```python
import subprocess
subprocess.run(["ffmpeg", "-i", "input.mp4", "output.mp3"])
```

---

## 3️⃣ 语音识别（核心）

### 推荐：OpenAI Whisper（API版）

👉 Python：

```python
from openai import OpenAI
client = OpenAI()

client.audio.transcriptions.create(
    model="whisper-1",
    file=open("audio.mp3", "rb")
)
```

---

### 可选增强（后期）

* faster-whisper（本地）
* FunASR（中文优化）

---

## 4️⃣ 声纹识别（单人检测核心）

### 推荐：pyannote.audio

👉 功能：

* 说话人分离（speaker diarization）

👉 判断逻辑：

* 只允许1个speaker

---

## 5️⃣ 音频分类（音乐过滤）

### 推荐组合：

#### 基础：

* librosa（音频分析）

```python
import librosa
```

#### 判断逻辑：

* 人声占比
* 能量分布
* 频谱特征

👉 简化方案（推荐先用）：

* Whisper结果为空 → 判定音乐
* 文本长度过短 → 剔除

---

# 🧠 三、关键工程建议（非常重要）

## 1. 先做最小闭环（MVP）

不要一开始全做：

👉 第一版只做：

* 输入链接
* 下载视频
* Whisper转写
* GPT生成Skill

---

## 2. 过滤模块后置

先不做声纹识别，先用：

* 文本长度过滤
* 简单规则

---

## 3. Prompt是核心资产

你这个产品真正的壁垒是：
👉 Skill生成Prompt

---

## 4. 成本控制

* Whisper按分钟收费
* GPT按token收费

👉 建议：

* 限制视频长度（如 ≤ 2分钟）
